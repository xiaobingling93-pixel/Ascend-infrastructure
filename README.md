# Infrastructure SIG介绍

本仓库用于托管Ascend开源社区基础设施团队的公开信息，包括不限于：会议日程，成员信息，服务文档和配置等信息

## 我们的愿景
Ascend开源项目基础设施聚焦为Ascend相关的开源项目提供易用，高效，安全的开发者和用户服务

### 我们的行为准则
为了更好，更高效的服务好Ascend昇腾开源和我们的用户：
- 我们会以[opensourceinfra](https://opensourceinfra.org/) 的理念治理Ascend昇腾开源的基础设施
- 我们的运作践行[four open 哲学](https://openinfra.org/four-opens/)
