# Ascend社区邮件列表指南

## 一、介绍
Ascend社区[邮件列表](https://mailweb.ascend.osinfra.cn/) 订阅和退订功能，您可以预定/退订感兴趣的邮件列表。

##  二、订阅邮件列表
### 浏览
直接从[邮件列表服务](https://mailweb.ascend.osinfra.cn/)进入，选择一个邮件进行订阅。

![](figures/list.png)

### 订阅
点击进入您感兴趣的组织，填写您的邮箱地址，名字（可选），点击订阅(Subscribe)。

![](figures/subscribe.png)

您需要前往您填写的邮箱进行个人信息确认，您的邮箱会收到如下邮件：

![](figures/subscribed-email.png)

您需要点击邮件里的确认链接，并点击Confirm按钮，即可成功订阅。
![](figures/subscribed-confirm.png)
![](figures/subscribed-success.png)


## 三、取消订阅
选择您想取消的邮件地址并点击进入。

![](figures/list.png)

输入您的邮箱地址后，点击取消订阅(Unsubscribe)。

![](figures/unsubscribe.png)

取消订阅后，您需要前往您填写的邮箱进行个人信息确认，您的邮箱会收到如下邮件：
![](figures/unsubscribe-email.png)

您需要点击邮件里的确认链接，即可成功取消订阅。
> 注意 取消订阅时不需要Confirm，点击链接即可成功取消订阅。


## 四、邮件列表的公共存档
邮件列表服务提供Archives模块，用来存放和查阅这个邮件列表所有历史邮件，您可以进入Archives模块查阅这个邮件列表所有历史邮件。

![](figures/archives-enter.png)

![](figures/archives-list.png)

## 五、发送邮件
您可以向[邮件列表服务](https://mailweb.ascend.osinfra.cn/)的邮箱地址发送邮件。
